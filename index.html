Pseudo-Selectors

Pseudo-selectors allow styling elements based on their state or position.

  • :hover - Styles when the mouse is over the element
  • :focus-visible - Styles when the element is focused via keyboard
  • :active - Styles when the element is being activated
  • :nth-child() - Selects elements by position
  • :not() - Excludes certain elements
  • :first-child - Selects the first child
  • ::before - Inserts content before the element
  • ::after - Inserts content after the element
Example code with pseudo-selectors: 
button:hover {
            background-color: #ff0000;
        }
        
        button:focus-visible {
            outline: 2px solid #0000ff;
        }

Responsive Design

Responsive design ensures that your layout adapts to different screen sizes using media queries.

Key breakpoints:

  • Mobile: 768px
  • Tablet: 768px - 1024px
  • Desktop: > 1024px
Media query example: 
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}
