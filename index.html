Pseudo-Selectores

Los pseudo-selectores permiten estilizar elementos basados en su estado o posición.

  • :hover - Estilos cuando el mouse está sobre el elemento
  • :focus-visible - Estilos cuando el elemento está enfocado vía teclado
  • :active - Estilos cuando el elemento está siendo activado
  • :nth-child() - Selecciona elementos por posición
  • :not() - Excluye ciertos elementos
  • :first-child - Selecciona el primer hijo
  • ::before - Inserta contenido antes del elemento
  • ::after - Inserta contenido después del elemento
Ejemplo de código con pseudo-selectores: 
button:hover {
    background-color: #ff0000;
}

button:focus-visible {
    outline: 2px solid #0000ff;
}

Diseño Responsivo

El diseño responsivo asegura que tu layout se adapte a diferentes tamaños de pantalla usando media queries.

Puntos de quiebre clave:

  • Móvil: 768px
  • Tablet: 768px - 1024px
  • Escritorio: > 1024px
Ejemplo de media query: 
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

Accesibilidad

Asegurar que tu sitio web sea accesible para todos los usuarios es crucial.

  • Usa elementos HTML semánticos
  • Proporciona texto alternativo para imágenes
  • Asegura suficiente contraste de color
  • Haz visibles los estados de foco
  • Usa estructura lógica de encabezados

Esta página incluye:

  • Jerarquía apropiada de encabezados
  • Soporte de navegación por teclado
  • Indicadores de foco visibles
  • Texto alternativo para imágenes decorativas
Notas de Diseño
  • Usé pseudo-selectores :hover, :focus-visible, :active, :nth-child(), :not(), :first-child, ::before y ::after
  • Implementé rayado de tabla con :nth-child(odd) y :nth-child(even)
  • Agregué efectos hover en filas de tabla y teclas de piano
  • Usé ::before y ::after para marcadores visuales en teclas de piano y elementos artísticos
  • Puntos de quiebre responsivos en 768px y 480px para móvil y tablet
  • Aseguré navegación por teclado con estados de foco visibles
